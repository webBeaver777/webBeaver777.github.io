Профиль

Я — Backend‑ориентированный Fullstack‑разработчик с 4+ годами коммерческой практики в PHP 8+ и Laravel 8+. Проектирую устойчивую архитектуру, пишу чистый код, ускоряю платформы за счёт оптимизации БД и грамотного использования кеша и очередей. Работал со стеком: MySQL/PostgreSQL, Redis, RabbitMQ, WebSockets, REST, на фронтенде — Vue 3 / Nuxt, Tailwind. Автоматизирую поставку через Docker и CI/CD, настраиваю Nginx/Caddy и окружения dev/stage/prod. Немного знаком с Go, писал Telegram‑ботов на Python (aiogram) для уведомлений, модерации и интеграций по вебхукам.

Ключевые технологии: PHP 8+, Laravel 8+, Vue 3, Nuxt, TypeScript, MySQL, PostgreSQL, Redis, RabbitMQ, Docker, CI/CD, Nginx, Caddy, Vite, Webpack, WebSockets, aiogram, Go (basic).
